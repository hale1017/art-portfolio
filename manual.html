網站維護手冊

這本手冊教你怎麼自己更新網站。全程不用碰任何程式, 就像填表單一樣 —— 放心亂試,任何東西改壞了都救得回來。

你的網站 回到首頁 管理後台(改內容都在這裡) /admin/

💡 建議把管理後台網址加到手機主畫面或瀏覽器書籤,平板上也很好用。這頁手冊的網址是 /manual.html,網站頁尾隨時點得到。

01登入後台

  1. 打開管理後台網址(上面那個「管理後台」連結)。
  2. 按「Sign in with GitHub」,用你的 GitHub 帳號授權。第一次會問一次,之後瀏覽器會記住。
  3. 看到左側出現「作品集、娃衣作品、材料包⋯」的選單,就是登入成功了。
忘記 GitHub 密碼? 到 github.com 按「Forgot password?」用信箱重設。 真的卡住就聯絡開發者,一定救得回來。

02最常做的事

新增一件畫作

  1. 左側選單點「作品集」→ 點進「作品列表」。
  2. 按「新增」,填:作品名稱、媒材分類、年份、尺寸、說明。
  3. 「作品圖片」按上傳,選照片(建議寬度 2000 像素以內,手機照片直接傳也可以)。
  4. 想放到首頁,打開「放到首頁精選」。
  5. 按右上角儲存/發布 → 等 10–30 秒 → 完成!

排列順序:在列表裡直接拖曳調整,網站會照同樣順序顯示。

新增一種媒材分類

例如想加「色鉛筆」——

  1. 左側選單點「媒材分類」,新增一筆。
  2. 代號(英文):小寫英文,例如 colored-pencil。設定好就不要再改。
  3. 顯示名稱:「色鉛筆」——這個隨時可以改。

儲存後,新增作品時就選得到「色鉛筆」,作品集頁的篩選按鈕也會自動長出來。

新增一件娃衣

  1. 左側選單點「娃衣作品」。
  2. 新增:成品名稱(例:小海豹喫茶)、系列(職業/動物)、介紹。
  3. 「照片」可以放很多張(正面、背面、上身照⋯),第一張會當封面,順序可拖曳。

新增 / 修改材料包

  1. 左側選單點「材料包」。
  2. 填:名稱、示範影片、封面圖、價格、訂購連結(蝦皮/賣貨便網址)。
影片規範:30MB 以內、大約 30–60 秒。手機拍好後,先用內建編輯功能剪短再上傳。
iPhone:照片 App → 選影片 → 編輯 → 拖動兩端剪裁。
Android:相簿 → 選影片 → 編輯 → 剪裁。

賣完了暫時沒貨?把「目前可訂購」關掉,網站會顯示「補貨中」,不用刪除。

改簡介・經歷・課程・聯絡方式

關於我
自我介紹直接打字,換行就會自動分段;經歷一條一條加,可拖曳排序。
教學課程
課程額滿就把「開放報名中」關掉;沒填報名連結會自動顯示「來信詢問」。
網站設定
網站標題、首頁大圖、Email、電話、IG/FB/LINE/蝦皮連結都在這裡。

03圖片與影片的小建議

東西建議
照片寬度 2000 像素以內、5MB 以下(手機原圖通常可以直接用)
影片30MB 以內、30–60 秒,直式橫式都可以
檔名用英文或數字比較保險(例:seal-cafe-1.jpg)

04改壞了怎麼辦?

完全不用怕。網站的每一次儲存都有完整紀錄,任何時間點的版本都能救回來。

  1. 先試試看自己改回來 —— 通常只是打錯字。
  2. 不行就聯絡開發者,說「幫我恢復到某天的樣子」即可,幾分鐘就能復原。

05常見問題

按了發布,網站沒變?

等 30 秒再「重新整理」;手機請下拉重整。還是沒變 → 過五分鐘再看一次 → 聯絡開發者。

後台介面是英文的?

外框按鈕是英文(Save、Publish 等),但所有欄位都是中文。記住兩個就好:Save/Publish = 儲存發布、Delete = 刪除。

可以在手機上用後台嗎?

可以,平板體驗更好。

影片太大傳不上去?

先照上面的方法剪短;還是太大就聯絡開發者幫忙壓縮。
